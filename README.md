# fastify_template_babaiyu
My Fastify Template

This template project created using:
1. Fastify
2. Typesript
3. Docker + Nginx

### Local Development
```
pnpm run dev
```

### Run Production
```
pnpm run clean-build && pnpm run build
pnpm run start
```

### Build & Running for Docker
```
docker-compose up --build -d
```

### Unit Test
```
pnpm run test
```
